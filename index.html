CrossTalk Client Patcher (C#)

Automatically patches legacy clients for CrossTalk servers.

========================================================

Status: Main branch is stable, .net4 (Xp,Vista,7) branch is a WIP

Currently supported patchable clients:

1. MSN Messenger (Binary, Registry, and Reroute/DLL Patches)

2. Yahoo! Messenger (Registry and Reroute/DLL Patches)


*Note on non-public support:

The CrossTalk team is currently working on adding support

for AIM/OSCAR and MySpaceIM. This is not yet public

and has been removed from this patcher build to prevent

issues. Please wait for an updated release.
